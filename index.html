Проекты

Веб-приложение "BBQ"

  • Приложение создано для организации мероприятий. BBQ дает возможность создавать мероприятия с указанием даты и места, чтобы другие пользователи могли сообщить организатору, что они собираются посетить мероприятие. Приложение содержит раздел комментариев для участников, чтобы обсудить событие, и фотогалерею, чтобы поделиться фотографиями.
  • Технологии: Ruby, Ruby on Rails, Для рассылки используется ActionMailer и Google почта, Yandex Cloud Storage для хранения фотографиями, Devise and Pundit для авторизации, OmniAuth через Yandex.

Веб-приложение "Who wants to be a billionaire"

  • Приложение с реализацией игры Кто хочет стать миллионером. Пользователи могут соревноваться в суммах которые они выиграли. Реализованы все 3 типа подсказок (звонок другу, 50/50, подсказка зала). Администратор может загружать новые в базу.
  • Технологии: Ruby, Ruby on Rails, Приложение покрыто тестами на модели, контроллеры и фича тестами, Реализация AJAX запроса.
